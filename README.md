# sandbox

_A sandbox environment with livereloading front-end, styles, and api_

Remember to branch before testing things out.

## Development

#### Getting started
 - `npm install`

#### Running environment (each has livereload)
 - `make web`     (web server at 8080)
 - `make api`     (api at 3000)
 - `make styler`  (styles)
